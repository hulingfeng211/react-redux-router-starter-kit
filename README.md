

# react-redux-router starter kit

> 这是一个基于react开发的Demo


- redux做应用状态框架
- react-router-redux作路由
- amazeui-touch做ui框架
- webpack做模块打包工具
- babel做es6和jsx编译
- gulp作为任务流管理器


### 介绍
> 这个demo实现了基本的app功能，以及时间旅行（time-travel）和热重载（react-hot-loader）等好玩功能

#### 安装依赖:
```
npm install
```

#### 普通运行
```
npm start
```

#### 开发调试
```
npm run dev
```

#### 打包
```
gulp build
```

#### 打开浏览器，访问

http://localhost:8080/m
